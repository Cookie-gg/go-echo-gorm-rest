# Go-echo-gorm-rest

REST API with golang echo and gorm

## Setup

1. make env file

```.env
MYSQL_ROOT_PASSWORD="drowssap"
MYSQL_DATABASE="development"
MYSQL_USER="dev"
MYSQL_PASSWORD="drowssap"
MYSQL_ROOT_HOST="%"
MYSQL_PORT=3306
MYSQL_HOST="db"
```

2. run docker-compose

```
docker compose up -d
```
